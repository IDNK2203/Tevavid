Rhea, Benin
Project Rhea is a project that merges both functionality and Sophistication, with the use of straight horizontal and vertical lines, we were able to harness basic shapes and achieve a very sophisticated and organised design that has all boxes checked. Coming with a lobby beside the house that has glazed large windows that can be slid open and turn the lobby into an open space. Beside the lobby is a swimming pool, this gives a refreshing feel to the end user whenever they pass through the lobby.
